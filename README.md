# FlashBTN


### [Check out the demo!!](https://dilionscode.github.io/FlashBtn/)


## How to use it?


Add the CSS file.

```
<link rel="stylesheet" href="flash.css">
```

Add the class!

```
<button class="flash hover">
```
List of animations

```
hover
slide-left
slide-right
slide-down
slide-up
```

Clone the code!
```
git clone https://github.com/DilionsCode/FlashBtn.git
```
