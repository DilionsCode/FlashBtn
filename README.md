# FlashBTN

## How to use it?


<p>Add the CSS file.</p>

```
<link rel="stylesheet" href="flash.css">
```

<p>How to use it.</p>

```
<button class="flash hover">
```
<p>Animations.</p>

```
hover
slide-left
slide-right
slide-down
slide-up
```
